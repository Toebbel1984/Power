A better windows shutdown dialog.

Usage:
    Power [OPTION]
Options:
    /reboot   (/r)   Startet den Computer neu.
    /shutdown (/s)   Fährt den Computer vollständig herunter.
    /lock     (/l)   Sperrt die aktuelle Benutzersitzung.
    /monoff   (/m)   Schaltet den Monitor aus.
    /option   (/o)   Restarted in das Boot Optionsmenü.
    /safe     (/v)   Restartet das System im abgesicherten Modus.
    /logoff   (/f)   Meldet den aktuellen Benutzer ab.
    /bios     (/b)   Restartet direkt ins BIOS/UEFI.
    /trouble  (/t)   Startet das system in den Troubleshooting modus.
Examples:
    Power /s
        → Fährt den Computer herunter.
    Power /r
        → Startet den Computer neu.
    Power /l
        → Sperrt den PC sofort.
Notes:
    - Kurzbefehle (Aliases) können anstelle der langen Befehle verwendet werden.
    - Alle Aktionen sind sofort wirksam. (Timeout = 0)
    - Verwenden Sie '/?' oder '/help' um diesen Hilfetext anzuzeigen.

Usage:
  Power [OPTION]
Options:
	/reboot   (/r) Restarts the computer.
	/shutdown (/s) Shuts down the computer completely.
	/lock     (/l) Locks the current user session.
	/monoff   (/m) Turns off the monitor.
	/option   (/o) Restarts to the boot options menu.
	/safe     (/v) Restarts the system in safe mode.
	/logoff   (/f) Logs off the current user.
	/bios     (/b) Restarts directly into BIOS/UEFI.
	/trouble  (/t) Starts the system in troubleshooting mode.
Examples:
  Power /s
    	→ Shuts down the computer.
  Power /r
  	  → Restarts the computer.
  Power /l
  	  → Locks the PC immediately.
Notes:
	- Shortcuts (aliases) can be used instead of the full commands.
	- All actions take effect immediately. (Timeout = 0)
	- Use '/?' or '/help' to display this help text.
